# Google Form Short Description

Memorable is a self-adapting memory layer for flight rebooking voice agents. We built it with LiveKit (voice runtime), Moss (retrieval + embeddings), TrueFoundry (routing), and MiniMax (primary routed model). In benchmark runs on the same cancellation prompt, cold mode retries restricted fare dead-ends and escalates, while memory mode replays a learned waiver→partner-flight→same-day policy workflow and resolves in one pass with fewer steps, lower token usage, and faster completion.
